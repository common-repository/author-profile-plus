Supercharge your WordPress user profiles with Author Profile Plus

== Description ==

Adds useful custom fields to your WordPress user profiles, allowing in depth user profiles to appear below the posts in your WordPress theme. Give your contributors the attention and credit they deserve with a featured biography and links to their social channels.

= Features =

* Easy to setup in just 2 simple steps.
* Seamlessly integrate author profiles throughout your WordPress website
* Supports multiple authors

If you have found this plugin useful, consider taking a moment to rate it.

== Installation ==

1. Upload the `author-profile-plus` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress

== Frequently asked questions ==

= How do I use the plugin? =

1. Install the plugin
2. Navigate to any user profile in the back end and fill out the Author Profile Plus fields
3. User Profiles should now appear below any post, or wherever the_content() is called
4. Alternatively, you can us the [author-profile] shortcode in your templates to display the author profiles wherever you like. Optionally, you can also pass in an ID, like so [author-profile id="5"]

== Upgrade notice ==

Upgrading to 0.8.2 will cause Author Profile Plus to only output author profiles on the 'post' post type. EG, blog posts. If your theme uses custom post types and you want Author Profile Plus to work with these as well, go to Settings -> Author Profile Plus and select which post types you need.
